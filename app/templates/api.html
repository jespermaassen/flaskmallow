General Information on Endpoints

- For GET endpoints, parameters must be sent as a query string.
- For POST, PUT, and DELETE endpoints, the parameters may be sent as a query string or in the request body with content type application/json.
You may mix parameters between both the query string and request body if you wish to do so.
- Parameters may be sent in any order.
- If a parameter is sent in both query string and request body, the query string parameter will be used.
